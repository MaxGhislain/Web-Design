Internal CSS

I prefer this type of CSS the most for the following reasons:

  • It is very intuitive
  • Very efficient and easy to use (definitly when compared to inline css)
  • The <style> tag allows for easy access, reading, and editing of html elements' css properties.
  • There is no need to use an external document.

Working with internal CSS removes all of the clutter inline CSS brings and allows for easy editing of multiple elements at a time! Internal CSS definitly looks much more elegant compared to Inline CSS.
